## Estructura
- `/`: Frontend (HTML/CSS/JS) para despliegue en Netlify.
- `/backend`: API (Node.js) para despliegue en Render.

## Despliegue

### Frontend (Netlify)
1. Arrastrar esta carpeta (excluyendo `backend` si es posible) a Netlify Drop.
2. O conectar este repositorio a Netlify.

### Backend (Render)
1. Conectar este repositorio a Render.
2. Configurar `Root Directory` como `backend`.
3. Definir variables de entorno (ver `backend/.env.example`).
